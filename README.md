# era-javascript-api

**era-javascript-api** is a simple node.js wrapper for ERA REST API.

# Install

    npm install poloniex-api-node

# Usage

### Era([options])

Example:

```js
var Era = require('era-javascript-api');
era = new Era()
	
era.lastBlock(function(data, err) {
  if (!err) console.log(data)
})
```
